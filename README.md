# Calories-Burnt-Prediction
This Project is about the Calories Burnt Prediction in Human Body using Machine Learning with Python by using  XGBoost Regressor model.
Calorie burnt prediction by machine learning algorithm” aim to predict the number of calories burnt by an individual during physical activity using machine learning techniques.
The dataset includes features such as heart rate, body temperature, and duration of activity.

Dataset Link - https://www.kaggle.com/datasets/fmendes/fmendesdat263xdemos
